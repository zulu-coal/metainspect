MetaInspect 🕵️‍♂️ (Universal Metadata Extractor & Privacy Cleaner)

MetaInspect is a lightweight, zero-server, 100% private universal metadata inspector and privacy cleaner. It extracts hidden diagnostics, embedded data models, and tracking elements from images, PDFs, and audio files—all directly inside the browser using secure local memory buffers.

✨ Key Features

Universal Parser Support:

Images: Decodes camera parameters, capture hardware, active focal parameters, and deep EXIF blocks.

Documents (PDF): Extracts authors, editor details, page counts, production software platforms, and edit histories.

Audio Tracks: Retrieves core playback metadata, sample configurations, and duration details safely.

Interactive Mapping: If your image contains hidden geographical coordinates, an integrated dynamic map instantly pinpoints the precise geolocation details.

One-Click Privacy Cleaner: Re-projects data onto clean canvas contexts to strip all metadata, allowing you to download a sanitized, tracking-free copy.

Ad Integration Ready: Fully responsive CSS structure with native layout boxes optimized for standard ad banners to monetize traffic easily.

🔒 Why Client-Side?

Most online file analyzers upload your personal documents and photos to remote servers to parse them, posing severe security and privacy risks.

MetaInspect runs entirely on your local machine. Your files are read into local JavaScript arrays inside your browser and never touch the network, keeping your data completely confidential and secure.

🚀 Quick Start & Deployment

Run Locally

Simply clone or download this repository, open the directory, and double-click the index.html file to run it in your browser immediately.
